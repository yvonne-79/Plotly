1.1 语言基础¶

1.1.1 标量类型¶

Python的标准库中拥有一个小的内建类型集合，用来处理数值数据、字符串、布尔值、日期和时间，这类“单值”类型称为标量类型。标准Python标量类型包括：

  • None：null值
  • str：字符串
  • bytes：原生ASCII字节
  • float：双精度64位浮点数
  • bool：布尔值，True或False
  • int：整数
  • datetime\date\time：使用标准库中的datetime模块

① 数值¶

整数 int 和浮点数 float

1.1.2 控制流¶

标准控制流在Python中通过一些条件逻辑、循环等内建关键字的方式实现。

① if 语句¶

if条件语句式最广为人知的控制流类型。一个if语句可以接多个elif代码块和一个else代码块。如果某个条件为True，则后面的elif和else代码块不会执行；如果所有elif条件均为False，则执行else代码块。

Python中的三元表达式允许将一个if-else代码块联合起来，在一行代码或一个语句中生成数据。使用三元表达式可以压缩代码量，但同时也会牺牲可读性。

In [20]:
x = 10
if x<0:
    print('It\'s negative')
else:
    print('It\'s positive!')

It's positive!
In [21]:
x = int(input())
if x<0:
    print('It\'s negative')
elif x==0:
    print('Equal to zero')
elif x<10:
    print('Positive but smaller than 10')
else:
    print('Positive but larger than or equal to 10')

8
Positive but smaller than 10
In [22]:
# 三元表达式
print('It\'s negative!') if x<0 else print('It\'s positive!')

It's positive!

④ 循环中的关键字¶

break关键字可以结束当前循环（如果多循环嵌套，则结束内层循环）；

continue关键字可以跳过continue之后的代码进入下一次循环；

else子句会在遍历完序列（for循环）或条件判断为False（while循环）时被执行，但不会在break终止时执行。

In [27]:
while True:
    input_str = input("Enter an integer, type quit if wannna stop ")
    if input_str == 'quit':
        break
    number = int(input_str)
    if number < 0:
        continue

    if number % 2 == 0:
        print(number,"is even")
    else:
        print(number,"is odd")

⑤ pass语句¶

就是Python中的“什么都不做语句”。当语法上需要一个语句（因为Python使用了缩进来分隔代码块），但实际上不需要执行任何操作时，可以使用pass语句。或者可以作为还没有实现的代码占位符。

In [29]:
# pass 
x=8
if x<0:
    print('It\'s negative!')
elif x==0:
    pass 
else:
    print('It\'s positive!')

It's positive!

1.2.6 推导式¶

推导式是很受欢迎的Python语言特性之一，允许过滤一些元素，以一种简明的表达方式生成一个新的序列。

① 列表推导式¶

[expr for val in collection if condition]

In [88]:
a_list = ['beer','wine','rum','coffee','tea','Coca Cola']
[v.upper() for v in a_list if len(v)>3]
Out[88]:
['BEER', 'WINE', 'COFFEE', 'COCA COLA']

② 集合推导式¶

{expr for val in collection if condition}

In [89]:
{len(v) for v in a_list}
Out[89]:
{3, 4, 6, 9}

③ 字典推导式¶

{key-expr:val-expr for val in collection if condition}

In [90]:
{v:i for i,v in enumerate(a_list) if len(v)>3}
Out[90]:
{'beer': 0, 'wine': 1, 'coffee': 3, 'Coca Cola': 5}

1.2.7 生成器 Generator¶

如果列表元素可以按照某种算法推算出来，那我们是否可以在循环的过程中不断推算出后续的元素呢？这样就不必创建完整的列表，从而节省大量的空间。在Python中，这种一边循环一边计算的机制，称为生成器Generator。生成器被用作控制循环的迭代行为，是迭代器的一种。

① 生成器表达式¶

使用生成器表达式来创建生成器，与列表推导式很类似，只需将中括号替换为小括号。

In [91]:
# 使用推导式创建一个列表
a_list = [i**2 for i in range(10)]
a_list
Out[91]:
[0, 1, 4, 9, 16, 25, 36, 49, 64, 81]
In [92]:
# 使用生成器表达式创建一个列表生成器
b_list = (i**2 for i in range(10))
b_list
Out[92]:
<generator object <genexpr> at 0x7fd4f02cba50>

② 生成器实例化¶

之前学习的内建序列函数很多也属于生成器，一般可以通过list()函数或for循环来进行实例化。

# 使用for循环来进行遍历
for i in b_list:
    print(i,end=' ')

1.3 函数、模块和包¶

1.3.1 函数Function¶

函数是Python中最重要、最基础的代码组织和代码复用方式。如果需要多次重复相同或类似的代码，就非常值得写一个可复用的函数。

函数声明时使用def关键字，后面跟函数名称和带括号的形参，返回时使用return关键字（允许多条return语句，如果达到函数尾部仍然没有遇到return语句则会自动返回None）。

① 定义和调用函数¶

In [97]:
# 函数声明时使用def关键字，后面跟函数名称和带括号的形参，返回时使用return关键字。
def fct(x,y,z=1):
    if z>=1:
        return z*(x+y)
    else:
        return x+1,y+2
In [98]:
# x,y是位置参数，z是关键字参数，用于指定默认值或可选参数，可以没有，有的话需在最后
fct(5,6)
Out[98]:
11
In [99]:
# 调用fct函数
fct(5,6,7)
Out[99]:
77
In [100]:
# 也可使用形参=实参的形式，有助于代码可读性
fct(x=5,y=6,z=7)
Out[100]:
77
In [101]:
# 返回多个值:即返回多个值组成的元组
fct(x=5,y=6,z=0.7)
Out[101]:
(6, 8)
In [102]:
# 元组可进一步做拆包
a,b=fct(x=5,y=6,z=0.7)

Out[102]:
8

② 匿名Lambda函数¶

匿名函数（Lambda）是一种通过单个语句生成函数的方式。

之所以被称为匿名函数，是因为允许没有一个显示的函数名称。定义匿名函数只需使用lambda关键字，后面跟形参：返回值。匿名函数在数据分析中经常会使用到，将它作为参数进行传值，代码量小，比写一个完整的函数更为方便。

In [103]:
# 定义Lambda函数只需使用lambda关键字，后面跟形参：返回值。
fct2 = lambda x:x**2
In [104]:
# 调用Lambda函数
fct2(5)
Out[104]:
25
In [105]:
# Lambda函数在数据分析中经常作为参数进行传值
seq = ['beer','wine','rum','coffee','tea','Coca Cola']

# 根据字符串的长度对列表进行排序
sorted(seq,key=lambda x:len(x))
Out[105]:
['rum', 'tea', 'beer', 'wine', 'coffee', 'Coca Cola']
In [106]:
# 根据字符串的字母顺序（忽略大小写）进行排序
sorted(seq,key=lambda x:x.lower())
Out[106]:
['beer', 'Coca Cola', 'coffee', 'rum', 'tea', 'wine']

1.3.2 模块 Module¶

模块就是包含Python定义和语句的，以.py作为后缀名的文件。

当你退出Python解释器后，之前的定义（类、函数、变量等）都会消失。或者当你想在不同的程序中使用一个函数，需要将函数复制到每一个程序中。Python提供了一种方法，将这些定义存储在被一个称为模块的独立文件中，再使用import语句将模块导入到主程序中，允许当前程序中使用模块中的代码。

1.3.3 包 Package¶

包可以看作是一个允许不断增长的模块的集合。

包通过“带点号的模块名”来对包中的模块名命，比如模块名A.B表示A包中的名为B的子模块。这样一来，来自不同包之间的模块就不会出现重名的情况。包的导入与模块的导入方式大致相同。

  1. 导入包package：
  • import package [as nickname] 使用全称/别称去访问item
  1. 导入包中特定的模块module：
  • import package[.subpackage].module 使用全称去访问item
  • from package[.subpackage] import module 使用模块名称去访问item
  1. 直接导入特定的item（类、函数、变量等）：
  • from package[.subpackage.module] import item 直接访问item
